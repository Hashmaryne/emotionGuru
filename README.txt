-----TO RUN BACKEND-----
1. Open project in preferred IDE
2. run: pip install -r requirements.txt
3. Open terminal cd and to project backend folder
4. Run project

-----TO RUN FRONTEND-----
1. Open project in preferred IDE
2. Open terminal and cd to project frontend folder
3. run: ionic serve --ssl
